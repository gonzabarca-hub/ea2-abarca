# Analizador Táctico de Pokémon (PokeAPI)

## Narrativa del Proyecto
* **Stakeholder:** Jugadores competitivos de Pokémon en emuladores o torneos.
* **Propuesta de Valor (Problema/Solución):** Durante una batalla, un jugador tiene segundos para decidir su próximo movimiento frente a un Pokémon rival. Esta herramienta de terminal permite consultar instantáneamente los stats de velocidad, ataque y HP para tomar decisiones tácticas rápidas sin tener que navegar por wikis lentas o pesadas.

## Guía de Configuración
Para ejecutar este proyecto, no se requieren llaves reales para la PokeAPI, pero para fines de arquitectura, se puede configurar la siguiente variable de entorno:
* Linux/Mac: `export API_KEY_PROYECTO="tu_clave_dummy"`
* Windows (PowerShell): `$env:API_KEY_PROYECTO="tu_clave_dummy"`
